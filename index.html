What we do

The world as our office.

Three founders walk 120 km of the Camino and build as we go. Laptop in the backpack, Starlink overhead, glasses for a screen. The point is how we run Claude Code: not with a keyboard and a monitor, but handsfree — voice in, a joystick to steer, the screen on our face. No new product, no custom agent. Just Claude Code in VS Code, off the desk, walking.

Where Claude comes in

We already build on Claude. The Camino is the extreme version.

We already depend on Claude Code for our daily build. We've seen it work on the move. The Camino is where we extend that to six days — with the desk, the keyboard and the screen taken away.

The fact
We build our website with Claude Code. We're already dependent on it for the daily build. That's true, that's now.
The proof
A friend built a working app on Claude Code while running a marathon. We're extending that to six days of walking.
The experiment
Whether Claude Code holds up with the desk, keyboard and screen gone — voice and a controller only. Heat, dead battery, six days.

Why MIND

MIND is the whole point.

What we build, we hand over to Stichting MIND for free — a Dutch mental-health foundation. The walk ends, the tool stays. That's the proof this isn't a stunt: something real comes out of it.

The ask

What we're asking.

No money, no sponsorship. Credits — and one person who can say yes.

What we need
$2,000 in API credits for the length of the project — enough to run Claude Code and the MIND tool while we push it to the limit. And one contact who can decide. We keep usage lean with prompt caching, so it isn't a blank cheque.
What you get
A live demonstration of Claude Code, run handsfree under the harshest conditions there are, with the result donated to a mental-health nonprofit. The "AI gives you your attention back" story — lived by three people, not claimed in an ad. Something Anthropic can't say about itself without it sounding like marketing.
The invitation

Push Claude to its limit with us.

We capture all of it in a short documentary, and the story opens up the moment there's footage. We'd love Anthropic in it from the first step.
